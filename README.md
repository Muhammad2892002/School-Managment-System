# School Management Module

This project is a module of a School Management System developed using **ASP.NET Core** and **Onion Architecture**.  
The module focuses on managing students, subjects, and recording student grades.

The goal of this project is to practice building a modular backend system with clean architecture and proper separation of concerns.

## Technologies Used

- ASP.NET Core
- C#
- Entity Framework Core
- SQL Server
- Onion Architecture
- Repository Pattern
- HTML / CSS / JavaScript

## Features

- Student registration and management
- Subject management
- Recording and updating student grades
- CRUD operations for system entities
- Database integration using Entity Framework
- Layered architecture separating Domain, Infrastructure, and UI

## Architecture

The project follows **Onion Architecture**, which separates the application into multiple layers:

- **Domain Layer** – Contains core entities and business models.
- **Infrastructure Layer** – Handles data access and repository implementations.
- **UI Layer** – Manages user interaction and presentation logic.

This structure helps maintain clean code, scalability, and easier maintenance.
